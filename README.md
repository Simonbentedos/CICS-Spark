# CICS SPARK Repository

A NextJS and NestJS SPARK (System for Preserving Academic Research and Knowledge) Web Application for the College of Information and Computing Sciences.

## Tech Stack

- Nest.js Frontend
- Next.js Backend
- PostgreSQL Database

## Getting Started

1. Install Dependencies
    - On both backend and frontend folders:    
```bash
npm install
```

2. Run the Development Server
```bash
npm run dev
```

3. Open
```text
Backend
http://localhost:5000

Frontend
http://localhost:3000
```

## Production Build

Build and Run locally:
```bash
npm run build
npm run start
```
Remember: do this on both folders.
